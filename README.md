### 製品説明（具体的な製品の説明）
　目的地を指定し、GPSで現在の位置を取得し、端末の向きを取得し、現在地から目的地までの直線の方角を示す

### 特長

#### 1. 特長1
　目的地までの方角だけを指定するので時間さえあればいつかは目的地に行ける
#### 2. 特長2
